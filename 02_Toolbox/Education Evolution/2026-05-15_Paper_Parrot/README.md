# 🛠️ Paper Parrot

> *Paper Parrot: Your Research Recall Riff-Raff*

---

**The problem:** Tired of reading research papers only to forget them? This tool helps you actively engage with texts to truly embed knowledge.

**What it does:** This tool extracts key arguments, methodologies, and findings from academic PDF papers and transforms them into structured questions and answer prompts, fostering active recall and deeper understanding for better knowledge retention.

**Born from:** ⚠️ `https://www.reddit.com/r/AskAcademia/comments/18z7g9c/how_do_you_keep_up_with_new_papers_without_losing/`

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. See all options
python main.py --help
```
