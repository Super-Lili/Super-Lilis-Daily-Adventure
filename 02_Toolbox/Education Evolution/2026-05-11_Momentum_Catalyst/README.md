# 🛠️ Momentum Catalyst

> *The Spark Starter: Reigniting Your Study Flame!*

**What it does:** This tool helps users overcome procrastination by guiding them to define tiny, actionable "ignition tasks" for large projects, providing timed focus sprints, and logging progress to build momentum.

**Born from:** ⚠️ [https://www.reddit.com/r/getdisciplined/comments/1cnv6q6/its_such_a_struggle_to_do_my_assignments_or_study/](https://www.reddit.com/r/getdisciplined/comments/1cnv6q6/its_such_a_struggle_to_do_my_assignments_or_study/)

## Quick Start
```bash
pip install -r requirements.txt
python main.py --help
```

## Run Tests
```bash
python test_main.py
```
