# 🛠️ Insight Loom

> *The Insight Loom: Weaving Clarity from Chaos*

---

**The problem:** Dive into your ideas! This tool helps you connect the dots in your learning, turning overwhelm into a beautifully mapped understanding.

**What it does:** This tool guides users to identify key concepts and define relationships between them from their notes or text, then saves the structured knowledge and a visual concept map.

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. See all options
python3 main.py --help
```

## Dependencies

```
pandas
networkx
matplotlib (optional, for visualization)
argparse
pathlib
```

## Run Tests
```bash
python3 test_main.py
```
