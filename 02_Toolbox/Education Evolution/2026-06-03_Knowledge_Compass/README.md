# 🛠️ Knowledge Compass

> *The Knowledge Compass: Your Curated Research Map*

---

**The problem:** Navigate complex topics with ease: get a meticulously curated 12-book reading list, organized by 6 distinct perspectives, ready to kickstart your deep dive.

**What it does:** This tool takes a research topic and generates a curated 12-book reading list, organized by six critical perspectives, for experienced researchers, journalists, and creative professionals.

**Born from:** ⚠️ GitHub Issue #2 by project owner | QUOTE: "开始研究一个新话题时，找书是最耗时的步骤——搜出来的推荐要么太泛、要么重复同一个角度、要么是 SEO 凑数书单。一个资深研究者需要的是：覆盖不同视角、真正有价值、可以立刻开始的书单。"

## Dependencies

```
This tool primarily uses standard Python libraries.
No external libraries are strictly required beyond what's usually available.
For a real-world implementation, an external API for book data (e.g., Google Books API)
and a sophisticated search engine integration would be necessary.
For this exercise, search results are simulated based on the prompt's requirements.
```
