# 🛠️ Variable Font Warm-Up Wheel

> *The Variable Font Warm-Up Wheel*

---

**The problem:** What if your font could warm up *with* you—not just for you?

**What it does:** A real-time, interactive wheel that maps two variable font axes (e.g., weight + width) to rotational input, with adjustable playback speed and built-in easing presets—so designers can physically *feel* axis relationships before animating.

**Born from:** ⚠️ Typewolf Forum | “Variable Fonts in Motion: A Practical Gap” (June 2026, pinned thread)
