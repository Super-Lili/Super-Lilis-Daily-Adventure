# 🛠️ Sun Light Color Clock

> *The Clock That Knows What Hour Feels Like*

---

**The problem:** Most clocks just count. This one responds to the quality of light at each moment of the day — without any configuration.

**What it does:** A full-screen browser clock whose background colour shifts through the full arc of natural daylight: deep indigo at night, warm gold at dawn, washed-out white at noon, coral at sunset, purple at dusk. Open it and leave it open.

**Born from:** GitHub Issue #1 by xiaojialove-DRP

---

## Quick Start

```bash
# Download
curl -O "https://raw.githubusercontent.com/Super-Lili/Super-Lilis-Daily-Adventure/main/02_Toolbox/Healing%20Inventions/2026-06-01_Sun_Light_Color_Clock/main.py"

# Run (generates sun_light_color_clock.html)

# Open in browser
open sun_light_color_clock.html
```
