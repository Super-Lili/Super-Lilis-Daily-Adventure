# 🛠️ Ambient Prompt Weaver

> *The Gentle Nudge Weaver: Crafting Calm Cues for Chaotic Days*

---

**The problem:** Tired of reminders that overwhelm? This tool creates subtle, ambient cues to guide your day without the visual chaos.

**What it does:** This tool takes a CSV file of tasks and their associated time blocks, then generates minimalist HTML "cue cards" for each block, displaying a single, gentle task reminder meant to be a quiet ambient prompt.
