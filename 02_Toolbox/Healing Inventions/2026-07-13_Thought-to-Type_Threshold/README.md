# 🛠️ Thought-to-Type Threshold

> *The Forgotten Half-Second Between Thought and Type*

---

**The problem:** A tool that doesn’t fix your focus — it gently stretches the half-second where thought becomes text.

**What it does:** A minimal, always-on HTML page that introduces a deliberate, adjustable 300–800ms delay between keypress and character appearance — turning the ‘frozen cursor’ moment into a perceptible, trainable threshold.

**Born from:** ⚠️ YouTube video: “How I Actually Learned Calculus in 3 Weeks (No Lectures)” — comment section, posted 2026-07-05, top pinned comment by u/chemlabghost
