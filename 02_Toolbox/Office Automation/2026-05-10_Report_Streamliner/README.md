# 🛠️ Report Streamliner

> *ReportStreamliner: Your Happy Data Harmonizer!*

**What it does:** This tool streamlines the process of combining multiple CSV or Excel files, aggregating data based on a common key, and generating a consolidated report.

**Born from:** ⚠️ [https://www.reddit.com/r/excel/comments/1cmq6r2/automating_excel_tasks_and_saving_report/](https://www.reddit.com/r/excel/comments/1cmq6r2/automating_excel_tasks_and_saving_report/)

## Quick Start
```bash
pip install -r requirements.txt
python3 main.py --help
```
