Gemini API 使用说明

模型优先级
gemini-2.5-flash
→ gemini-2.0-flash
→ gemini-2.0-flash-lite
重试策略
每个模型最多 3 次
退避间隔：15s · 30s · 60s
共 9 次总尝试机会
Google Search Grounding
brain.py 启用，为每日工具提供实时信源
每日 1–2 次调用，远低于免费配额上限
可用性保障
每天 3 个备用触发时间槽
短暂限流由重试逻辑覆盖
全天宕机（极罕见）则本周跳过，优雅降级
